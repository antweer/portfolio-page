Salesforce


EventWorks is an event management application that integrates the Salesforce REST API for database management and administration with a Angular frontend through a Node.js backend built on Express. Event planners can add events to be displayed on this app along with details such as event descriptions, event sessions, start/end times, seat availability, and more through the Salesforce website. Event goers can use the UI to browse through events that are listed and register for events and sessions by providing a name, email address, phone number, and company name. Once registered, an email will be sent to event goers via Salesforce to confirm registration.

MongoDB

Heroku


Bulleted is a note-taking application with storage and realtime synchronization across multiple devices. Bulleted at its core is a Singe Page Application with a Meteor backend and React-Redux frontend. Meteor's built in SocketIO library is leveraged for Bulleted's realtime synchronization. Bulleted uses MongoDB to store and save user's notes. Other features that Bulleted has are: account creation (with email validation), sorting by date, and SSL encryption via Heroku.

Node.js

Express

HTML5

CSS3

PostgreSQL

nginx


ScoreHoard is a RESTful API that tracks scores for videogame makers. ScoreHoard allows gamemakers to send scores (and the usernames associated with the scores) directly into the ScoreHoard servers, quickly being integrated into our custom database. The ScoreHoard API allows you to receive your aggregate data in easily usable JSON. Features include: GET requests for retrieving leaderboards from our database, POST requests for adding gamescores into our database, user login and account creation system made from scratch with email validation for authentication, randomized and unique API key generation, and the ability to have scores tracked for multiple games through a single account.

Ember Alert

django

JavaScript

HTML5

CSS3

PostgreSQL

nginx


Ember Alert tracks NASA satellite data for active fires happening around the world. Our interactive map allows you to check if there are any fires near you. Ember Alert also allows you to submit your geolocation data to us along with a phone number or an email address so that we can notify you if we ever detect a fire close by.


Ember Alert holds a special place in my heart since it was my first ever web application. I created it at the Houston NASA Space Apps Challenge with only 2 weeks of development experience under my belt. It was daunting and challenging to compete in a hackathon with such little experience, but my perseverance paid off because my group won the People's Choice award! I'm currently working on refactoring Ember Alert with Django.
